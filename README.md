# Modelo Conceitual de E-commerce

Este projeto apresenta o modelo conceitual de um sistema de e-commerce.

O diagrama contempla as seguintes entidades:
Cliente (Pessoa Física e Pessoa Jurídica)
Pedido
Pagamento
Entrega
O objetivo é representar as principais relações de um sistema de vendas online.
Ferramenta utilizada: diagrams.net (draw.io)
